# ship-landing checklist

Use this checklist after generating or updating the landing page.

## Core sections
- hero section exists
- feature section exists
- FAQ section exists
- CTA section exists
- SEO metadata is included

## Page quality
- page structure is clear
- section order is logical
- placeholder copy is clearly marked
- CTA is visible and understandable
- content is easy to edit later

## Project safety
- unrelated pages were not broken
- missing images or brand assets are clearly marked
- manual editing areas are documented
- styling stays consistent with the project

## Final review
- explain what was added
- explain what still needs manual editing
- explain how to review the landing page
