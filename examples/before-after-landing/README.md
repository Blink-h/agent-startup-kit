# before-after-landing

This example explains what a typical project may look like before and after using the `ship-landing` skill.

## Before

A typical project before using this skill may have:
- no clear hero section
- no structured feature section
- no FAQ section
- no clear CTA section
- missing SEO metadata
- inconsistent landing page structure

## After

After using the `ship-landing` skill, the project should usually have:
- a clear hero section
- a structured feature section
- an FAQ section
- a clear CTA section
- basic SEO metadata
- a more consistent landing page structure

## What this example is for

This example helps users quickly understand the practical value of the skill before reading the full skill files.
